# Класс для работы с файлами

Класс позволяет удобно работать с файлами и содержит все базовые методы

#### Использованые библиотеки:

```
os
shutil
```

## Основные методы:

- **create() - создание файла**
- **write(content) - запись в файл чего-либо**
*аргумент content содержит текст который запишут в файл*
- **read() - чтение файла (возвращает содеримое файла)**
- **delete() - удаление файла**
- **rename(name,extension) - переименовывание файла**
*аргумент name содержит новое имя файла, а аргумент extension содержит новое разширение файла*
- **copy(path) - копирование файла**
*аргумент path содержит путь до копируемого файла*


#### init
Аргументы:
- file_name - имя файла, с которым планируете работать
- file_extension - разширение файла, с которым планируете работать
- create_now_or_not - саздание файла сразу или нет(без вызова функции 'create()')
